## Aplikasi ini dibuat dengan :
1. PHP Version 7.3.11
2. CodeIgniter-3.1.11-0
3. SMTP Gmail
4. Admin template AdminBSB
5. Landing Page template MDBootstrap
6. free images support by freepik.com

## Instruction
1. copy folder ini di folder htdocs jika menggunakan xampp, atau di public_html server web
2. setting database di "application/config/database", sesuaikan dengan pengaturan
3. buka settingan phpmyadmin di browser kemudian import database.
4. buka browser, ketikkan alamat sesuai yang dilakukan di langkah nomor 1. *misal : localhost/nama-folder (jika di localhost)
5. Aplikasi siap digunakan

## Additional
1. wajib setting SMTP Gmail di folder "controllers/auth" function _email_reset dan _email_sukses sebelum menggunakan aplikasi online
2. jika icon pada tampilan web tidak muncul itu disebabkan karena tidak sedang terkoneksi dengan internet.
